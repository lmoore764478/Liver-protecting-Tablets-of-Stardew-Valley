# Pelican Town Planner · 星露谷信息助手

面向 **「时间不充裕，但有信息差」** 的星露谷玩家：帮你 **少查 Wiki、少白跑图**，在有限时间里高效获得成就感——**不是替你玩的作弊器**。

📄 **完整产品方案**：[docs/PRODUCT.md](./docs/PRODUCT.md)

---

## 产品定位

```text
缩短查攻略时间  +  避免无效跑图  =  有限时间内的最大成就感
```

**三条原则**

1. **Player executes, tool plans** — 玩家操作，工具规划  
2. **Defaults, not dictates** — 给建议，不强制唯一最优  
3. **Respect the cozy loop** — 保留探索与养成感  

---

## 功能一览

| 模块 | 说明 | 状态 |
|------|------|------|
| 收菜日历 | 播种记录 → 自动算成熟日 | ✅ v1 |
| 今日动线 | 收菜 → 浇水 → 任务，带顺序与耗时 | ✅ v1 |
| 时间预算 | 15 / 30 / 60 分钟优先级裁剪 | ✅ v1 |
| 进度同步 | 引导设置 +「下一天」+ 第一年阶段推断 | ✅ v1 |
| 🎯 目标拆解 · 还缺啥 | 搜「马厩」→ 材料/金币 + 背包缺口 | ✅ v2 |
| ⏰ 换季预警 | 今天不种就来不及 + 并入今日动线 | ✅ v2 |
| 📅 事件提醒 | 节日 / 生日 / 猪车 / 酱料女皇 + 浏览器通知 | ✅ v2 |
| 🗺️ NPC 地图 · 送礼 | 日程推算位置 + 礼物喜好 + 示意图 | ✅ v2.1 |
| 📈 进度可视化 | 社区中心进度条 + 下一步建议 | ✅ v2.1 |
| 🤫 防剧透模式 | 探索模式开关，隐藏心事件/剧情奖励 | ✅ v2.1 |
| 📂 存档导入 | 上传 .zip/.xml 自动读日期/金币/背包 | ✅ v2.1 |

**明确不做**：自动跑图、键鼠宏、代打、实时控制游戏。

---

## 游戏进度怎么跟踪？

| 方式 | 说明 | 版本 |
|------|------|------|
| 引导 +「下一天」 | 对照游戏日期，玩完一天点一下 | v1 ✅ |
| 手动勾选进度 | 社区中心收集包勾选 | v2.1 ✅ |
| 存档导入 | 解析 save 文件，自动读日期/金币/材料 | v2.1 ✅ |
